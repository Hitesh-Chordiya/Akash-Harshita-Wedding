
# Akash & Harshita Wedding Website

## How to Use

1. Replace images inside `/assets`
   - couple-main.jpg
   - photo1.jpg
   - photo2.jpg
   - photo3.jpg
   - photo4.jpg

2. Add your music file:
   - assets/music.mp3

3. Run locally:
   - Just open `index.html`

4. Deploy on Vercel
   - Push to GitHub
   - Import project in Vercel
   - Deploy

No build step required.
